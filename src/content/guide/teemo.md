**티모(Teemo)**는 리그 오브 레전드에서 가장 사랑받으면서도 가장 미움받는 챔피언입니다. "악마의 화신"이라는 별명처럼, 실명과 독 버섯으로 상대를 고통스럽게 만드는 것을 즐기는 요들 전사입니다.

---

## 티모는 어떤 챔피언인가?

티모는 밴들 시티의 신속한 정찰대원으로, 귀여운 외모와는 달리 매우 잔인한 전투 스타일을 가지고 있습니다. 실명과 독 피해, 그리고 버섯 지뢰로 상대 탑 라이너를 미치게 만듭니다.

### 기본 스탯 및 정보

| 항목 | 내용 |
|------|------|
| 역할군 | 마크스맨 / 암살자 |
| 포지션 | 탑 |
| 공격 유형 | 원거리 |
| 특징 | 실명 + 독 + 버섯 |
| 난이도 | ★★★☆☆ |

### 현재 메타에서의 위치 (패치 24.24)

- **승률**: 51.2%
- **픽률**: 3.8%
- **밴률**: 6.5%

> 원거리 견제로 근접 챔피언을 괴롭히는 전문 카운터 픽입니다.

## 스킬 상세 분석

### ![패시브](/images/champions/teemo/passive.png) 패시브 - 게릴라 전술

가만히 있으면 **투명** 상태가 됩니다.

**효과:**
- 1.5초간 가만히 있으면 투명
- 투명 해제 후 공격 속도 증가
- 부시 안에서는 이동 중에도 투명 유지

기습과 부시 컨트롤에 활용하는 유틸 패시브입니다.

---

### ![Q](/images/champions/teemo/q.png) Q - 실명 다트

적에게 **실명**을 부여합니다.

**효과:**
- 대상 지정 피해 + 실명 (1.5/1.75/2/2.25/2.5초)
- 피해량: 80/125/170/215/260 (+AP 80%)
- 쿨타임: 8초

AA 의존 챔피언을 무력화하는 핵심 스킬입니다.

---

### ![W](/images/champions/teemo/w.png) W - 신속 이동

**이동 속도**가 증가합니다.

**효과:**
- 패시브: 이동 속도 10/14/18/22/26% 증가
- 활성화: 이동 속도 2배로 증가 (3초간)
- 피해를 받지 않을 때만 패시브 적용
- 쿨타임: 17초

추격과 도주에 사용하는 기동성 스킬입니다.

---

### ![E](/images/champions/teemo/e.png) E - 맹독 쐐기

기본 공격에 **독 피해**를 추가합니다.

**효과:**
- 패시브: 기본 공격에 독 피해 추가
- 4초간 독 피해 지속
- 즉시 피해: 14/25/36/47/58 (+AP 30%)
- 지속 피해: 24/36/48/60/72 (+AP 40%)

티모의 핵심 딜 스킬로, 지속적인 독 피해를 줍니다.

---

### ![R](/images/champions/teemo/r.png) R - 악마의 버섯

**독 버섯**을 설치합니다.

**효과:**
- 지정 위치에 버섯 설치 (투명 상태)
- 적이 밟으면 폭발 + 독 피해 + 둔화
- 버섯은 5분간 지속
- 최대 3/4/5개 보유 (시간에 따라 충전)
- 피해량: 200/325/450 (+AP 55%) (4초간)
- 둔화: 30/40/50%

맵 컨트롤과 시야 확보에 사용하는 시그니처 스킬입니다.

## 스킬 콤보 가이드

### 기본 견제 콤보
**AA → AA → AA (E 독 스택)**
기본 공격으로 독 피해 누적

### Q 교환 콤보
**Q → AA → AA → AA**
Q 실명 후 AA 연타

### 올인 콤보
**W → AA → Q → AA × n**
W 접근, Q 실명, AA 연타

### 버섯 콤보
**R (길목) → 적 도착 → Q → AA × n**
버섯 밟은 적 추격

### 기습 콤보
**패시브 (투명) → AA → Q → AA × n**
투명에서 기습

## 추천 룬 세팅

> **3줄 요약**
> 1. 핵심 룬은 흑마법의 일격으로 지속 피해를 강화한다
> 2. 부 룬은 정밀으로 공격 속도를 보완한다
> 3. 비전 혜성도 포킹에 좋다

| 티어 | 룬 | 설명 |
|------|-----|------|

| 1티어 | 마나순환 팔찌 | 마나 지속력 |
| 2티어 | 절대 집중 | 70% 체력 이상 시 AP |
| 3티어 | 주문 작열 | 잃은 체력 비례 피해 |

### 대체 핵심 룬

|-----|------|
| **흑마법의 일격** | 지속 딜 / 표준 |

| **집중 공격** | 교환 중심 |

### 부 룬: 정밀

| 룬 | 설명 |

### 능력치 파편

## 추천 아이템 빌드

> **3줄 요약**
> 1. 코어는 나쉬르의 이빨 → 리안드리의 고뇌 → 그림자불꽃 순서로 간다
> 2. 신발은 광전사의 군화가 기본이다
> 3. 온힛 AP 빌드가 주류이다

### 코어 빌드

| 순서 | 아이템 | 가격 |
|------|--------|------|
| 시작템 | **도란의 반지 + 물약** | 500G |
| 신발 | **광전사의 군화** | 1100G |
| 1코어 | **나쉬르의 이빨** | 3000G |
| 2코어 | **리안드리의 고뇌** | 3000G |
| 3코어 | **그림자불꽃** | 2800G |

공격 속도 + AP로 독 피해 극대화.

### 신발 선택

| 신발 | 상황 |
|------|------|
| **광전사의 군화** (60%) | 기본 선택 |
| **마법사의 신발** (30%) | AP 극대화 |
| **수은의 장화** (10%) | CC 많은 상대 |

### 상황별 아이템

| 아이템 | 상황 |
|--------|------|
| **존야의 모래시계** | 암살자 상대 |
| **모렐로노미콘** | 힐 챔피언 상대 |
| **공허의 지팡이** | 마저 상대 |
| **라바돈의 죽음모자** | 딜 극대화 |
| **밴시의 장막** | AP 버스트 상대 |

### 최종 풀빌드
광전사의 군화 → 나쉬르의 이빨 → 리안드리의 고뇌 → 그림자불꽃 → 라바돈의 죽음모자 → 존야의 모래시계

## 라인전 운영 가이드

> **3줄 요약**
> 1. 원거리 우위로 근접 챔피언을 괴롭힌다
> 2. Q 실명으로 적의 반격을 막는다
> 3. 버섯으로 갱킹을 대비한다

### 레벨 1~5
- **레벨 1**: E로 AA 견제
- **레벨 2**: Q로 실명 교환
- **레벨 3**: W로 기동성 확보
- **원거리 우위**: 근접 챔피언 완전 압도

### 레벨 6
- R 버섯으로 갱킹 대비
- 강 입구와 부시에 버섯 설치
- 킬 확정력 상승

### 1코어 이후
- 나쉬르로 AA 딜 상승
- 적극적인 견제와 킬 시도
- 버섯으로 맵 컨트롤

### 라인전 팁
1. **AA 견제**: 끊임없이 AA
2. **Q 타이밍**: 적 AA 시 Q로 실명
3. **버섯 설치**: 갱킹 경로에 버섯
4. **W 도주**: 갱킹 시 W로 탈출

---

## 버섯 설치 가이드

> **3줄 요약**
> 1. 버섯은 시야와 맵 컨트롤의 핵심이다
> 2. 갱킹 경로와 오브젝트 근처에 설치한다
> 3. 라인 한가운데에도 설치하여 적 CS 방해

### 추천 설치 위치
1. **강 입구**: 갱킹 경로
2. **트라이부시**: 갱킹 경로
3. **정글 입구**: 침입 감지
4. **미드 로밍 경로**: 로밍 감지
5. **오브젝트 근처**: 드래곤/바론

### 버섯 활용법
- 시야 확보
- 갱킹 대비
- 도주로 차단
- 웨이브 클리어

## 한타 운영 가이드

> **3줄 요약**
> 1. 티모는 팀파이트에서 후방 딜러/유틸 역할을 한다
> 2. Q로 적 ADC를 실명시킨다
> 3. 버섯으로 진입로를 막는다

### 역할: 후방 딜러 / 유틸

**기본 전략**: 버섯으로 진입로 차단 → Q 실명 → AA 딜

1. 팀파이트 전 버섯 설치
2. 후방에서 안전하게 딜
3. 적 ADC에게 Q 실명
4. W로 리포지셔닝

### 포지셔닝
**좋음**: 후방, 안전한 거리
**나쁨**: 최전방, 고립된 위치

### Q 실명 우선순위
1. 적 ADC
2. 공속 의존 챔피언
3. 다른 딜러

## 카운터 정보

> **3줄 요약**
> 1. 티모는 갭클로저와 버스트가 있는 챔피언에게 약하다
> 2. AA 의존 근접 챔피언에게 강하다
> 3. 상대할 때는 갭클로저로 접근 후 원콤한다

### 티모가 어려운 상대

| 챔피언 | 이유 |
|--------|------|
| **판테온** | 점사 + 스턴 |
| **이렐리아** | 갭클로저 + 올인 |
| **요네** | 갭클로저 + 버스트 |
| **아칼리** | 점사 + 은신 |
| **야스오** | W로 Q 막음, 갭클로저 |

### 티모가 쉬운 상대

| 챔피언 | 이유 |
|--------|------|
| **가렌** | 접근 불가, 실명 |
| **다리우스** | 접근 불가, 카이팅 |
| **세트** | 접근 불가, 카이팅 |
| **나서스** | 접근 불가, Q 파밍 방해 |
| **트린다미어** | 실명, 카이팅 |

### 티모 상대법
1. **갭클로저**: 접근 후 원콤
2. **버섯 주의**: 이동 시 버섯 확인
3. **올인**: 짧은 교환 금지
4. **정글 도움**: 갱킹으로 압박

## 마무리

티모는 귀여운 외모와 달리 상대를 미치게 만드는 악마의 챔피언입니다. 실명과 독, 버섯으로 근접 챔피언을 완전히 무력화할 수 있으며, 맵 컨트롤 능력도 뛰어납니다. 다만 팀파이트에서는 존재감이 약할 수 있습니다.

> **티모를 한 줄로 요약하면:**
> "크기가 중요한 게 아니야! (버섯 밟으며 사망)"
