**문도 박사(Dr. Mundo)**는 리그 오브 레전드에서 가장 대표적인 체력 탱커입니다. "문도는 가고 싶은 데로 간다"라는 대사처럼, CC를 무시하고 엄청난 체력 회복으로 죽지 않는 것이 특징입니다.

---

## 문도 박사는 어떤 챔피언인가?

문도 박사는 자운의 미친 의사로, 자기 자신을 실험 대상으로 삼아 끔찍한 실험을 계속하고 있습니다. 마나가 없고 체력으로 스킬을 사용하며, 엄청난 체력 회복 능력을 가지고 있습니다.

### 기본 스탯 및 정보

| 항목 | 내용 |
|------|------|
| 역할군 | 전사 / 탱커 |
| 포지션 | 탑 / 정글 |
| 공격 유형 | 근거리 |
| 특징 | 마나 없음 + 체력 회복 |
| 난이도 | ★★☆☆☆ |

### 현재 메타에서의 위치 (패치 24.24)

- **승률**: 50.8%
- **픽률**: 4.2%
- **밴률**: 3.5%

> 단순한 조작과 강력한 생존력으로 초보자에게 추천되는 탱커입니다.

## 스킬 콤보 가이드

### 기본 견제 콤보
**Q → AA**

### 교환 콤보
**Q → E → AA → W (방출)**
Q 둔화 후 E 강타, W로 회복

### 올인 콤보
**Q → R → E → AA → W → E → AA**
R 회복 + 이속으로 풀콤보

### 갱킹 콤보 (정글)
**Q → E → AA → Q → AA**
Q 둔화로 접근, E-AA 딜

### 다이브 콤보
**R → Q → E → AA → 퇴각**
R 체력으로 다이브

## 추천 아이템 빌드

> **3줄 요약**
> 1. 코어는 정령의 형상 → 워모그의 갑옷 → 선혈포식자 순서로 간다
> 2. 신발은 판금 장화 또는 수은의 장화가 기본이다
> 3. 체력과 회복 아이템 위주로 빌드한다

### 코어 빌드

| 순서 | 아이템 | 가격 |
|------|--------|------|
| 시작템 | **도란의 방패 + 물약** | 450G |
| 신발 | **판금 장화** | 1100G |
| 1코어 | **정령의 형상** | 2800G |
| 2코어 | **워모그의 갑옷** | 3200G |
| 3코어 | **선혈포식자** | 3200G |

체력과 회복력을 극대화하는 빌드.

### 신발 선택

| 신발 | 상황 |
|------|------|
| **판금 장화** (50%) | AD 상대 |
| **수은의 장화** (40%) | CC 많은 상대 |
| **신속의 장화** (10%) | 기동성 필요 시 |

### 상황별 아이템

| 아이템 | 상황 |
|--------|------|
| **가시 갑옷** | AD 상대 |

| **자연의 힘** | AP 상대 |
| **갈망의 투구** | 흡혈 강화 |

### 최종 풀빌드
판금 장화 → 정령의 형상 → 워모그의 갑옷 → 선혈포식자 → 가시 갑옷 → 자연의 힘

## 라인전 운영 가이드

> **3줄 요약**
> 1. Q로 견제하며 안전하게 CS를 챙긴다
> 2. 패시브로 CC를 무효화한다
> 3. 레벨 6 이후 R로 생존력이 대폭 상승한다

### 레벨 1~5
- **레벨 1**: Q로 CS + 견제
- **레벨 2**: E로 교환력 상승
- **레벨 3**: W로 회복력 확보
- **패시브 활용**: CC 무효화

### 레벨 6
- R로 생존력 대폭 상승
- 다이브 당해도 R로 생존
- 솔킬 능력도 상승

### 1코어 이후
- 정령의 형상으로 마저 + 회복
- 적극적인 교환
- 팀파이트 참여 가능

### 라인전 팁
1. **Q 견제**: 꾸준히 Q 투척
2. **패시브 체력통**: 반드시 회수
3. **E 사용**: CS에도 E 활용 가능
4. **R 타이밍**: 저체력 시 R

## 한타 운영 가이드

> **3줄 요약**
> 1. 문도는 팀파이트에서 프론트라인/다이버 역할을 한다
> 2. R로 죽지 않으면서 적 캐리에게 접근한다
> 3. 패시브로 핵심 CC를 무효화한다

### 역할: 프론트라인 / 다이버

**기본 전략**: R 활성화 → 적 캐리에게 돌진 → 딜

1. 패시브로 첫 CC 무효화
2. R로 체력 회복
3. Q로 적 캐리 둔화
4. E로 지속 딜

### 포지셔닝
**좋음**: 최전방, 적 캐리 근처
**나쁨**: 후방 (딜이 없음)

### R 사용 타이밍
1. 팀파이트 시작 시
2. 체력 50% 이하일 때
3. 다이브 직전

## 카운터 정보

> **3줄 요약**
> 1. 문도는 힐 감소나 고정 피해가 있는 챔피언에게 약하다
> 2. 체력 스케일링이 약한 챔피언에게 강하다
> 3. 상대할 때는 힐 감소 아이템을 필수로 간다

### 문도가 어려운 상대

| 챔피언 | 이유 |
|--------|------|
| **베인** | 고정 피해, 카이팅 |
| **피오라** | 고정 피해, 힐감 |
| **릴리아** | 카이팅, 힐감 |
| **그웬** | 고정 피해 |
| **요릭** | 지속 딜, 분리 |

### 문도가 쉬운 상대

| 챔피언 | 이유 |
|--------|------|
| **말파이트** | 딜 부족 |
| **오른** | 딜 부족 |
| **마오카이** | 딜 부족 |
| **초가스** | 비슷한 스케일링 |
| **쉔** | 딜 부족 |

### 문도 상대법
1. **힐 감소**: 필수 아이템
2. **고정 피해**: 체력 탱커에 효과적
3. **체력통 밟기**: 패시브 회복 방해
4. **카이팅**: 이동기 없는 문도 카이팅
