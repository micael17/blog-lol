**비에고(Viego)**는 리그 오브 레전드에서 가장 강력한 정글러 중 하나입니다. 적을 처치하고 빙의하는 독특한 메카닉으로 팀파이트에서 연쇄 킬을 만들어내는 캐리 정글러입니다.

---

## 비에고는 어떤 챔피언인가?

비에고는 '몰락한 왕'으로 불리는 그림자 제도의 지배자입니다. 적 챔피언을 처치하면 그 영혼에 빙의하여 해당 챔피언의 스킬을 사용할 수 있는 독특한 패시브를 가지고 있습니다.

### 기본 스탯 및 정보

| 항목 | 내용 |
|------|------|
| 역할군 | 전사 / 암살자 |
| 포지션 | 정글 (미드/탑 가능) |
| 공격 유형 | 근접 |
| 특징 | 빙의 + 리셋 |
| 난이도 | ★★★★☆ |

### 현재 메타에서의 위치 (패치 24.24)

- **승률**: 50.8%
- **픽률**: 11.2%
- **밴률**: 18.5%

> 높은 밴률이 증명하는 강력한 캐리력을 가진 정글러입니다.

## 스킬 콤보 가이드

### 기본 갱킹 콤보
**E (안개) → W (충전) → Q → 평타 → R**
안개로 접근, W 기절 후 딜, R로 마무리

### 풀 콤보 (킬각)
**W (최대 충전) → Q → 평타 → 평타 → R**
W 기절로 이니시, Q와 평타로 체력 깎고 R 마무리

### 팀파이트 리셋 콤보
**Q → R (처치) → 빙의 → 해당 챔피언 스킬 사용 → R (빙의 해제) → 반복**
처치 후 빙의, R로 다음 타겟, 연쇄 킬

### 안개 기습 콤보
**E → 접근 → W (충전 없이) → Q → 평타**
안개로 기습, 빠른 딜링

### 마무리 콤보
**R (저체력 적에게) → 빙의 → E → 도주**
R로 마무리 후 빙의해서 안전하게 도주

## 추천 룬 세팅

> **3줄 요약**
> 1. 핵심 룬은 정복자로 지속 교전에서 회복과 딜을 높인다
> 2. 부 룬은 지배로 처치 시 회복과 궁극기 가속을 챙긴다
> 3. 빠른 정글링을 원하면 칼날비도 선택 가능

| 티어 | 룬 | 설명 |

## 추천 아이템 빌드

> **3줄 요약**
> 1. 코어는 몰락한 왕의 검 → 삼위일체 → 방어템 순서로 간다
> 2. 몰락한 왕의 검은 Q 패시브와 시너지가 매우 좋다
> 3. 상황에 따라 치명타 빌드나 탱킹 빌드로 변형 가능

### 코어 빌드

| 순서 | 아이템 | 가격 |
|------|--------|------|
| 정글템 | **모스토퍼** | 350G |
| 신발 | **판금 장화 / 수은 장화** | 1100G |
| 1코어 | **몰락한 왕의 검** | 3200G |
| 2코어 | **삼위일체** | 3333G |
| 3코어 | **죽음의 무도** | 3300G |

몰락한 왕의 검의 현재 체력 비례 피해가 Q 패시브와 겹쳐 매우 강력합니다.

### 신발 선택

| 신발 | 상황 |
|------|------|
| **판금 장화** (60%) | AD 상대 |
| **수은의 장화** (35%) | CC/AP 상대 |
| **광전사의 장화** (5%) | 스노우볼 시 |

### 상황별 아이템

| 아이템 | 상황 |
|--------|------|
| **수호 천사** | 집중 타겟 시 |
| **정령의 형상** | AP 상대 |
| **스테락의 도전** | 버스트 대응 |
| **마무리의 속삭임** | 탱커 상대 |
| **무한의 대검** | 치명타 빌드 |

### 치명타 빌드 (스노우볼용)
광전사의 장화 → 크라켄 학살자 → 몰락한 왕의 검 → 무한의 대검 → 수호 천사

### 최종 풀빌드 (기본)
판금 장화 → 몰락한 왕의 검 → 삼위일체 → 죽음의 무도 → 정령의 형상 → 수호 천사

## 정글 클리어 가이드

> **3줄 요약**
> 1. 레드 시작으로 체력 관리하며 풀클리어 한다
> 2. Q 패시브로 큰 몬스터를 빠르게 처리한다
> 3. E 안개로 갱킹 경로를 숨기며 접근한다

### 클리어 루트

**기본 루트 (레드 시작):**
레드 → 크루그 → 랩터 → 늑대 → 블루 → 그롬프 → 갱킹/스컷

**블루 시작 루트:**
블루 → 그롬프 → 늑대 → 랩터 → 레드 → 크루그 → 갱킹/스컷

### 클리어 팁
1. **Q 패시브 활용**: 큰 몬스터에 Q 패시브 피해 집중
2. **체력 관리**: 레드 버프로 회복하며 클리어
3. **스마이트 타이밍**: 큰 몬스터에 스마이트로 빠른 클리어

---

## 갱킹 및 운영 가이드

> **3줄 요약**
> 1. E 안개로 갱킹 경로를 숨기고 기습한다
> 2. W 충전 기절이 핵심이므로 정확한 타이밍에 사용한다
> 3. 레벨 6 이후 R 마무리로 킬 확률이 높아진다

### 갱킹 방법
1. **E 안개 활용**: 벽을 따라 안개 생성, 위장 상태로 접근
2. **W 기절**: 최대 충전으로 긴 기절
3. **Q + 평타**: 딜링
4. **R 마무리**: 저체력 적에게 확정 킬

### 오브젝트 컨트롤
- **드래곤**: Q 패시브로 빠른 처치
- **헤럴드**: W 기절로 아군과 연계
- **바론**: 팀과 함께, R로 스틸 시도

### 운영 팁
1. **킬 우선**: 비에고는 킬로 스노우볼하는 챔피언
2. **적 정글 침입**: 1:1에서 강하므로 적 정글 장악
3. **팀파이트 타이밍**: 먼저 들어가지 말고 빙의 기회 노리기

## 한타 운영 가이드

> **3줄 요약**
> 1. 비에고는 팀파이트 중반에 진입하여 리셋을 노린다
> 2. 저체력 적에게 R로 마무리하고 빙의로 다음 타겟을 노린다
> 3. 빙의 중 적 챔피언의 강력한 스킬을 활용한다

### 역할: 캐리 / 암살자

**기본 전략**: 탱커가 이니시 후 저체력 적에게 R로 진입, 빙의로 연쇄 킬

1. 팀파이트 시작 시 후방에서 대기
2. 적 체력이 낮아지면 R로 진입
3. 처치 후 빙의 → 해당 챔피언 스킬 활용
4. R 초기화 → 다음 타겟 → 반복

### 빙의 우선순위
1. **강력한 CC 챔피언**: 레오나, 노틸러스 등
2. **고딜러**: 원딜, 마법사
3. **기동성 챔피언**: 도주용

### 포지셔닝
**좋음**: 팀파이트 측면, 저체력 적 노릴 수 있는 위치
**나쁨**: 최전방 이니시, 적 CC 사거리 내

## 카운터 정보

> **3줄 요약**
> 1. 비에고는 강한 초반 정글러와 CC가 많은 팀에게 약하다
> 2. 빙의 전에 터지면 리셋이 불가능하다
> 3. 상대할 때는 비에고가 진입하면 집중 화력으로 터뜨린다

### 비에고가 어려운 상대

| 챔피언 | 이유 |
|--------|------|
| **리 신** | 강한 초반 정글 장악 |
| **엘리스** | 초반 갱킹 압박 |
| **렝가** | 기습으로 한 방에 터짐 |
| **카직스** | 고립 시 버스트 |
| **라마스** | CC + 탱킹으로 접근 차단 |

### 비에고가 쉬운 상대

| 챔피언 | 이유 |
|--------|------|
| **마스터 이** | 초반 약함, 1:1 유리 |
| **케인** | 변신 전 약함 |
| **이블린** | 레벨 6 전 약함 |
| **피들스틱** | 1:1 불리 |
| **아무무** | 1:1 불리, 정글 침입에 약함 |

### 비에고 상대법
1. **초반 침입**: 비에고 정글 침입으로 성장 방해
2. **CC 집중**: 진입 시 CC로 빙의 전에 처치
3. **분산**: 한 명씩 죽지 않도록 주의
4. **시야 장악**: 비에고 위치 파악

## 마무리

비에고는 적을 처치하고 빙의하는 독특한 메카닉으로 팀파이트에서 무한 리셋이 가능한 캐리 정글러입니다. 높은 스킬캡으로 숙련이 필요하지만, 마스터하면 혼자서 게임을 캐리할 수 있습니다.

> **비에고를 한 줄로 요약하면:**
> "한 명 죽이면 두 명 죽이고, 두 명 죽이면 다 죽인다"
