**징크스(Jinx)**는 리그 오브 레전드에서 가장 상징적인 원딜 챔피언입니다. 아케인 애니메이션의 주인공으로 더욱 유명해진 징크스는 후반 팀파이트의 여왕으로 불립니다.

---

## 징크스는 어떤 챔피언인가?

징크스는 자운 출신의 미친 범죄자로, 두 가지 무기(미니건과 로켓 런처)를 사용합니다. 처치/어시스트 시 폭발적인 이동 속도를 얻어 연속 처형이 가능합니다.

### 기본 스탯 및 정보

| 항목 | 내용 |
|------|------|
| 역할군 | 마크스맨 |
| 포지션 | 원딜 |
| 공격 유형 | 원거리 |
| 특징 | 후반 캐리 + 리셋 |
| 난이도 | ★★★☆☆ |

### 현재 메타에서의 위치 (패치 24.24)

- **승률**: 51.5%
- **픽률**: 15.2%
- **밴률**: 8.5%

> 후반 팀파이트에서 압도적인 캐리력을 보여주는 하이퍼 캐리입니다.

## 스킬 상세 분석

### ![패시브](/images/champions/jinx/passive.png) 패시브 - 신난다!

처치/어시스트 시 **폭발적인 이동 속도**를 얻습니다.

**효과:**
- 적 처치/어시스트 또는 구조물 파괴 시 발동
- 175% 이동 속도 증가 (점점 감소)
- 6초간 공격 속도 대폭 증가

팀파이트에서 연속 처형의 핵심입니다.

---

### ![Q](/images/champions/jinx/q.png) Q - 무기 전환! (파우파우/피시본)

두 무기 사이를 **전환**합니다.

**효과:**
- **파우파우 (미니건)**:
  - 공격 시 공격 속도 스택 (최대 3스택)
  - 빠른 연사로 DPS 극대화
- **피시본 (로켓 런처)**:
  - 사거리 증가 (+80/110/140/170/200)
  - 범위 피해
  - 마나 소모

라인전은 미니건, 팀파이트는 로켓이 기본입니다.

---

### ![W](/images/champions/jinx/w.png) W - 철컥!

전기충격을 발사하여 **피해 + 둔화**를 줍니다.

**효과:**
- 긴 사거리 스킬샷
- 적중 시 피해 + 70% 둔화 (2초)
- 피해량: 10/60/110/160/210 (+AD 160%)
- 쿨타임: 8/7/6/5/4초

적 추격이나 도주 시 필수 스킬입니다.

---

### ![E](/images/champions/jinx/e.png) E - 불꽃놀이!

바닥에 **덫**을 설치합니다.

**효과:**
- 3개의 덫 설치 (5초간 지속)
- 적이 밟으면 속박 + 피해
- 속박 시간: 1.5초
- 피해량: 70/120/170/220/270 (+AP 100%)
- 쿨타임: 24/20.5/17/13.5/10초

도주로 차단이나 셀프 필링에 사용합니다.

---

### ![R](/images/champions/jinx/r.png) R - 초대형 로켓!

**맵 전체**를 가로지르는 거대한 로켓을 발사합니다.

**효과:**
- 맵 전체 사거리
- 첫 번째 적 챔피언에게 적중 시 폭발
- 거리에 따라 피해량 증가 (최대 +35%)
- 잃은 체력 비례 추가 피해 (최대 25~35%)
- 쿨타임: 90/75/60초

저체력 적 처형이나 다른 라인 지원에 활용합니다.

## 스킬 콤보 가이드

### 기본 딜 패턴
**Q (미니건) → AA × 3 → Q (로켓)**
미니건으로 스택 쌓고 로켓으로 전환

### 추격 콤보
**W → AA → E → AA**
W 둔화 후 추격, E로 속박

### 킬 확정 콤보
**E → W → AA → R**
속박 후 풀콤보

### 팀파이트 패턴
**Q (로켓) → AA × n → (패시브 발동) → 추격**
로켓으로 딜, 패시브로 리셋

### 글로벌 저격
**R (예측)**
맵 전체에서 저체력 적 저격

## 추천 룬 세팅

> **3줄 요약**
> 1. 핵심 룬은 치명적 속도로 DPS를 극대화한다
> 2. 부 룬은 마법으로 마나와 성장을 보완한다
> 3. 집중 공격도 강력한 대안이다

### 주 룬: 정밀

| 티어 | 룬 | 설명 |
|------|-----|------|
| 핵심 | **치명적 속도** | 공격 시 공격 속도 증가 |
| 1티어 | 침착 | 처치 시 마나 회복 |
| 2티어 | 전설: 민첩함 | 강인함 획득 |
| 3티어 | 최후의 일격 | 저체력 적에게 추가 피해 |

### 대체 핵심 룬

| 룬 | 상황 |
|-----|------|
| **치명적 속도** | DPS 극대화 / 표준 |
| **집중 공격** | 버스트 + 지속력 |

### 부 룬: 마법

| 룬 | 설명 |
|-----|------|
| **절대 집중** | 70% 체력 이상 시 적응형 |
| **폭풍의 결집** | 후반 AP/AD 추가 |

### 능력치 파편
공격 속도 / 적응형 능력치 / 체력

## 추천 아이템 빌드

> **3줄 요약**
> 1. 코어는 크라켄 학살자 → 환상의 무희 → 인피니티 에지 순서로 간다
> 2. 신발은 광전사의 군화가 기본이다
> 3. 후반에는 공격 속도와 치명타를 쌓는다

### 코어 빌드

| 순서 | 아이템 | 가격 |
|------|--------|------|
| 시작템 | **도란의 검 + 물약** | 500G |
| 신발 | **광전사의 군화** | 1100G |
| 1코어 | **크라켄 학살자** | 3100G |
| 2코어 | **환상의 무희** | 2800G |
| 3코어 | **인피니티 에지** | 3400G |

크라켄으로 DPS, 환상의 무희로 기동성 확보.

### 신발 선택

| 신발 | 상황 |
|------|------|
| **광전사의 군화** (90%) | 기본 선택 |
| **판금 장화** (7%) | AD 암살자 상대 |
| **수은의 장화** (3%) | CC 많은 상대 |

### 상황별 아이템

| 아이템 | 상황 |
|--------|------|
| **도미닉 경의 인사** | 탱커 상대 |
| **피바라기** | 생존력 필요 시 |
| **수호 천사** | 암살자 상대 |
| **모렐로노미콘** | 힐 챔피언 상대 |
| **루난의 허리케인** | 범위 딜 강화 |

### 최종 풀빌드
광전사의 군화 → 크라켄 학살자 → 환상의 무희 → 인피니티 에지 → 도미닉 경의 인사 → 수호 천사

## 라인전 운영 가이드

> **3줄 요약**
> 1. 미니건으로 스택을 유지하며 CS를 챙긴다
> 2. W로 적을 견제하고 E로 갱킹을 대비한다
> 3. 초반은 약하니 안전하게 플레이한다

### 레벨 1~5
- **레벨 1**: 미니건으로 CS, 스택 유지
- **레벨 2**: W로 적 견제
- **레벨 3**: E로 갱킹 대비
- **초반 약함**: 무리한 교환 자제

### 레벨 6
- R로 다른 라인 지원 가능
- 저체력 적 마무리
- 솔킬 능력은 여전히 약함

### 1코어 이후
- 크라켄으로 DPS 상승
- 미니건 스택 + 로켓으로 교환
- 팀파이트 참여로 스노우볼

### 라인전 팁
1. **미니건 스택**: 항상 3스택 유지
2. **로켓 사거리**: 안전 거리에서 로켓
3. **E 위치**: 도주로에 미리 설치
4. **R 저격**: 다른 라인 저체력 확인

## 한타 운영 가이드

> **3줄 요약**
> 1. 징크스는 팀파이트에서 하이퍼 캐리 역할을 한다
> 2. 로켓으로 범위 딜을 하고, 패시브로 리셋한다
> 3. 첫 킬 후 연속 처형이 핵심이다

### 역할: 하이퍼 캐리

**기본 전략**: 후방에서 로켓 딜 → 패시브 발동 → 연속 처형

1. 안전한 후방에서 로켓으로 딜
2. 첫 킬/어시스트 획득
3. 패시브 이동 속도로 추격
4. 연속 처형으로 팀파이트 장악

### 포지셔닝
**좋음**: 후방, 안전한 거리
**나쁨**: 최전방, 고립된 위치

### 패시브 활용
1. 첫 킬 후 적극적 추격
2. 이동 속도로 리포지셔닝
3. 공격 속도로 DPS 극대화

## 카운터 정보

> **3줄 요약**
> 1. 징크스는 돌진 암살자나 초반 강세 원딜에게 약하다
> 2. 후반 스케일링이 약한 챔피언에게 강하다
> 3. 상대할 때는 초반에 압도하거나 후반 접근을 막아야 한다

### 징크스가 어려운 상대

| 챔피언 | 이유 |
|--------|------|
| **드레이븐** | 초반 압도 |
| **트리스타나** | 돌진으로 압박 |
| **사미라** | 돌진 + 궁극기 |
| **루시안** | 초반 강세, 기동성 |
| **칼리스타** | 기동성으로 압박 |

### 징크스가 쉬운 상대

| 챔피언 | 이유 |
|--------|------|
| **코그모** | 비슷한 후반 캐리, 이동기 없음 |
| **바루스** | 이동기 없음 |
| **애쉬** | 비슷한 스케일링 |
| **미스 포츈** | 후반 약함 |
| **질리언** | 원딜 아님, 위협 적음 |

### 징크스 상대법
1. **초반 압도**: 징크스 초반 약함
2. **돌진**: 징크스 이동기 없음
3. **E 유도**: E 쿨타임 때 공격
4. **패시브 방지**: 첫 킬 주지 않기

## 마무리

징크스는 후반 팀파이트의 여왕으로, 패시브 리셋으로 연속 처형이 가능한 하이퍼 캐리입니다. 초반은 약하지만 후반에는 압도적인 DPS로 게임을 캐리할 수 있습니다.

> **징크스를 한 줄로 요약하면:**
> "한 명 죽이면, 다 죽인다"
